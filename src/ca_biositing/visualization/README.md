# ca_biositing.visualization

This package contains visualization and reporting tools for the CA Biositing
project. It provides utilities for generating charts, maps, and interactive
reports from the project's data models.

## Installation

This package is managed as part of the `ca-biositing` workspace using Pixi. It
is included in the `viz` environment.

```bash
pixi run -e viz python -c "import ca_biositing.visualization"
```
